使用Python 3.8.2 和 beautifulsoup4 
從Yahoo Finance搜尋和分析美股option，找出擁有過大交易量的合約。


This program using Python 3.8.2 and beautifulsoup4 to build. 
It is going to read the Yahoo Finance options.

It will find the option that is traded extremely. 
Since most of the options end on Friday, it only shows the options end on Friday.
